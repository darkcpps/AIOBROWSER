CreamAPI Installation Instructions
==================================

CreamAPI files are not included with this application and must be obtained manually.

REQUIRED FILES:
--------------
Place the following files in this directory (tools/creamapi/):
  - steam_api.dll (for 32-bit games)
  - steam_api64.dll (for 64-bit games)

WHERE TO GET CREAMAPI:
---------------------
CreamAPI is distributed through community forums and websites. 
You can find it by searching for "CreamAPI" on:
  - auto-creamapi-2 repository: https://github.com/Warwolfer/auto-creamapi-2
  - Community gaming forums
  - creamapi.org (information site)
  - Various GitHub repositories (search for "creamapi" on GitHub)

NOTE: The application will attempt to automatically download CreamAPI DLLs
from the auto-creamapi-2 repository. If that fails, you can manually place
the files in this directory.

IMPORTANT:
----------
- Only use CreamAPI if you have the right to modify game files
- Using DLC unlockers may violate terms of service
- Use at your own risk

INSTALLATION:
------------
1. Download CreamAPI from a trusted source
2. Extract the archive
3. Copy steam_api.dll and/or steam_api64.dll to this folder
4. The patcher will automatically detect and use these files
